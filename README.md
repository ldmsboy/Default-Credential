## 🇪🇸 Descripción de Versión (Español)
**🔖 v1.0 (Lanzamiento Inicial):**
La primera versión de este proyecto, construida puramente con HTML estático, Tailwind CSS implementado a través de CDN, y JavaScript Vanilla. Todo funciona dentro de un único archivo donde se conecta directamente al servicio de la nube Firebase Firestore ("Frontend-as-Backend" approach). Permite realizar búsquedas ágiles, filtrado por Nivel de Riesgo CVSS y copiado automatizado al portapapeles.

## 🇬🇧 Version Description (English)
**🔖 v1.0 (Initial Release):**
The foundational version of this project, purely built using static HTML, CDN-loaded Tailwind CSS, and Vanilla JavaScript. Everything is powered within a single environment establishing direct links to the Firebase Firestore cloud backend ("Frontend-as-Backend" approach). It supports agile queries, CVSS Risk Level filtering, and automated clipboard-copying mechanisms.
